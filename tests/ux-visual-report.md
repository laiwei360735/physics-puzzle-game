### 2.4 无障碍设计评估

#### 颜色对比度

**WCAG 2.1 AA 标准要求：**
- 正常文字：对比度 ≥ 4.5:1
- 大文字（≥18pt）：对比度 ≥ 3:1
- UI 组件：对比度 ≥ 3:1

**对比度检测：**

| 元素组合 | 前景色 | 背景色 | 对比度 | 标准 | 结果 |
|----------|--------|--------|--------|------|------|
| 主按钮文字 | #FFFFFF | #FFA500 | 2.8:1 | 3:1 | ⚠️ 不达标 |
| 关卡数字 | #FFFFFF | 渐变绿 | 3.2:1 | 3:1 | ✅ 勉强达标 |
| UI 顶部栏文字 | #FFFFFF | rgba(0,0,0,0.5) | 5.1:1 | 4.5:1 | ✅ 达标 |
| 提示面板文字 | #555555 | #FFE4B5 | 3.8:1 | 4.5:1 | ⚠️ 不达标 |

**问题：**
1. ⚠️ 主按钮白字橙底对比度不足，视力障碍用户可能看不清
2. ⚠️ 提示面板文字对比度偏低

**建议：**
```css
/* 方案 1：加深文字颜色 */
.text-button {
  color: #333333;  /* 原 #FFFFFF → 深灰色 */
}

/* 方案 2：加深背景颜色 */
.btn-primary {
  background: linear-gradient(180deg, #FF8C00 0%, #E67E00 100%);
}
```

#### 文字大小

**设计系统字号：**
```css
--font-size-xs: 14px;   /* ⚠️ 偏小 */
--font-size-sm: 18px;   /* ✅ 适合正文 */

--font-size-lg: 32px;   /* ✅ 适合小标题 */
--font-size-xl: 36px;   /* ✅ 适合大标题 */
```

**评估：**
| 场景 | 字号 | 可读性 | 建议 |
|------|------|--------|------|
| 按钮文字 | 24px | ✅ 良好 | 保持 |
| 关卡数字 | 72px | ✅ 优秀 | 保持 |
| 正文说明 | 18px | ✅ 良好 | 保持 |
| 辅助文字 | 14px | ⚠️ 偏小 | 提升至 16px |
| 加载文字 | 24px | ✅ 良好 | 保持 |

**无障碍评分：** 6/10
- ✅ 主要 UI 元素字号充足
- ⚠️ 颜色对比度多处不达标
- ❌ 无字体大小调节选项
- ❌ 无色盲模式

**建议：**
1. 在设置中添加"大字体模式"（+20% 字号）
