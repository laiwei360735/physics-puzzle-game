**代码位置:** `src/scenes/GameScene.ts` (第 260-360 行)

**关键技术:**
```typescript
// 切割检测核心逻辑
private checkCutRopeIntersection(start, end): void {
  // 遍历所有绳子约束
  for (let constraint of playerBody.ropeConstraints) {
    // 检测线段相交
    if (this.lineIntersection(start, end, ropeStart, ropeEnd)) {
      // 移除约束
      this.matter.world.removeConstraint(constraint);
    }
  }
}
```

**验收结果:** ✅ 切割功能流畅（60fps），绳子正确分离

collectStar(star: StarData): void {
  // 1. 播放金色粒子特效
  this.vfxManager.createCollisionParticles(star.x, star.y, 15, 0xffd700);
  
  // 2. 星星消失动画
  this.scene.tweens.add({
    targets: star.gameObject,
    scale: 1.5,
    alpha: 0,
    rotation: Math.PI * 2,
