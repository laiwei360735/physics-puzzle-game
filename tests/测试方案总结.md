## 🚀 使用指南

### 安装依赖

```bash
cd /root/.openclaw/workspace/projects/physics-puzzle-game/tests
npm install
```

### 运行测试

```bash
# 运行所有测试
npm test

# 运行单元测试
npm run test:unit

# 运行集成测试
npm run test:integration

# 生成覆盖率报告
npm run test:coverage
```

### 查看测试报告

```bash
# 打开覆盖率 HTML 报告
open coverage/index.html
```

## ⚠️ 注意事项

1. **微信环境模拟**: 测试代码中模拟了微信小游戏的全局对象（`wx`），实际运行需要使用微信开发者工具或真机

2. **性能测试**: 性能测试需要在真机上运行才能获得准确数据

3. **兼容性测试**: 建议使用云测平台（Testin、WeTest）覆盖更多设备

4. **持续集成**: 建议将测试集成到 CI/CD 流程中，每次提交自动运行
