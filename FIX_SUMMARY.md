## 🎮 游戏玩法

### 操作方式

1. **点击玩家** - 开始拖拽
2. **拖动玩家** - 控制移动方向
3. **释放** - 玩家按物理规律运动
4. **到达目标** - 关卡完成

### 关卡元素

| 元素 | 说明 | 首次出现 |
|------|------|----------|
| 玩家 | 绿色圆形，可拖拽 | 关卡 1 |
| 目标 | 红色圆形，到达即通关 | 关卡 1 |
| 平台 | 灰色矩形，障碍物 | 关卡 2 |
| 绳子 | 悬挂的物体 | 关卡 4 |
| 风扇 | 产生风力 | 关卡 6 |
| 弹簧 | 弹性物体 | 关卡 8 |
| 磁铁 | 吸引/排斥 | 关卡 10 |

### 星级评价

- ⭐ 完成关卡
- ⭐⭐ 达到标准分数
- ⭐⭐⭐ 达到最高标准（步数限制）

## 🚀 部署说明

### 本地开发

```bash
# 克隆仓库
git clone https://github.com/laiwei360735/physics-puzzle-game.git
cd physics-puzzle-game

# 安装依赖
npm install

# 开发模式
npm run dev

# 访问 http://localhost:3000
```

### 生产构建

```bash
# 构建
npm run build

# 预览
npm run preview

# 部署 dist/ 目录到服务器
```

### 微信小游戏

```bash
# 构建微信版本
npm run build:wechat

# 使用微信开发者工具导入 dist/ 目录
```
