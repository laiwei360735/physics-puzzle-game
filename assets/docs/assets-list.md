## 🔧 实现建议

### SVG 使用

1. 直接嵌入 HTML 使用
2. 作为背景图：`background-image: url('xxx.svg')`
3. 作为 img 标签：`<img src="xxx.svg">`

### CSS 动画

1. 添加类名触发动画：`element.classList.add('bounce-celebrate')`
2. 监听动画结束：`element.addEventListener('animationend', callback)`
3. 移除动画类：动画结束后移除类名以便重播

### 响应式适配

1. 使用相对单位（rem、%）
2. 媒体查询断点：768px
3. 微信小程序使用 rpx 单位
