# 物理益智微信小游戏 - 游戏设计文档

**项目名称：** 物理大师 (Physics Master)  
**版本：** 1.0

## 一、游戏概述

### 1.1 核心理念

一款结合物理模拟与益智解谜的休闲小游戏，玩家通过操控物理元素（重力、碰撞、流体、切割等）解决各种创意谜题。游戏融合《割绳子》的物理交互乐趣与《倒水排序》的颜色分类挑战，创造独特的游戏体验。

### 1.2 目标用户

- **核心用户：** 18-35 岁，休闲游戏爱好者
- **次要用户：** 学生群体、通勤族、碎片时间玩家
- **用户特征：** 喜欢动脑、享受解谜成就感、愿意为优质内容付费

### 1.3 游戏特色

| 特色 | 描述 |
|------|------|
| 🎯 物理真实 | Matter.js 引擎提供真实物理模拟 |
| 🧩 创意谜题 | 每关独特设计，无重复解法 |
| 🎨 精美画风 | 清新卡通风格，视觉舒适 |
| 📱 即点即玩 | 微信小游戏，无需下载 |
| 🏆 挑战无限 | 每日挑战 + 关卡编辑器（后期） |

## 二、核心玩法机制

### 2.1 基础交互

#### 2.1.1 操作方式

| 操作 | 手势 | 效果 |
|------|------|------|
| 切割 | 单指滑动 | 切断绳子/线条 |
| 拖动 | 长按拖拽 | 移动可动物体 |
| 点击 | 单次点击 | 触发机关/按钮 |
| 双击 | 快速双击 | 特殊技能激活 |
| 双指缩放 | 双指开合 | 视图缩放（部分关卡） |

#### 2.1.2 物理元素

**可交互对象：**

```
┌─────────────────────────────────────────────────────┐
│  元素类型          │  物理属性        │  交互方式    │
├─────────────────────────────────────────────────────┤
│  绳子/链条         │  可切割、可摆动  │  滑动切割    │
│  球体/方块         │  重力、碰撞      │  拖动/点击   │
│  液体 (水/油/酸)   │  流动、混合      │  倾斜容器    │
│  弹簧/弹床         │  弹性反弹        │  碰撞触发    │
│  风扇/气流         │  风力推动        │  开关控制    │
│  磁铁              │  吸引/排斥       │  极性切换    │
│  传送带            │  定向移动        │  速度调节    │
│  炸弹/爆炸物       │  范围冲击波      │  点燃触发    │
└─────────────────────────────────────────────────────┘
```

### 2.2 胜利条件

- **主要目标：** 将目标物体送达指定位置/状态
- **星级评价：**
  - ⭐ 1 星：完成关卡
  - ⭐⭐ 2 星：收集额外物品（如星星、金币）
  - ⭐⭐⭐ 3 星：在限定步数/时间内完成

### 2.3 失败条件

- 目标物体掉落出屏幕
- 目标物体被危险物破坏（酸液、尖刺等）
- 超出最大操作次数
- 超时（限时关卡）

### 3.2 经典模式 (Classic Mode)

**描述：** 主线剧情关卡，循序渐进学习游戏机制

**特点：**
- 5 大主题世界，每世界 20 关，共 100+ 关卡
- 剧情驱动：帮助小物理学家完成实验
- 难度曲线平滑，逐步引入新元素
- 三星收集系统

**主题世界：**

| 世界 | 主题 | 核心元素 | 关卡数 |
|------|------|----------|--------|
| 1 | 森林实验室 | 绳子、重力、基础碰撞 | 20 |
| 2 | 水上乐园 | 液体、浮力、容器 | 20 |
| 3 | 工厂流水线 | 传送带、机械臂、开关 | 20 |
| 4 | 太空站 | 低重力、磁铁、真空 | 20 |
| 5 | 魔法学院 | 传送门、时间暂停、特殊道具 | 20 |

### 3.3 挑战模式 (Challenge Mode)

**描述：** 特殊规则的高难度挑战，每日更新

**子模式：**

#### 3.3.1 限时挑战 (Time Attack)
- 在限定时间内完成关卡
- 全球排行榜竞争
- 每日 3 个随机关卡

#### 3.3.2 步数挑战 (Move Master)
- 限定操作次数完成关卡
- 考验最优解法
- 每周更新题库

#### 3.3.3 一命通关 (One Cut)
- 只能切割一次/操作一次
- 极高难度，适合硬核玩家
- 成就系统关联

#### 3.3.4 盲盒挑战 (Mystery Box)
- 随机物理参数（重力方向、摩擦力等）
- 每次体验不同
- 娱乐性强

### 3.4 创意工坊 (Workshop Mode) - 二期功能

**描述：** 玩家自制关卡分享平台

**功能：**
- 关卡编辑器（拖拽式）
- 关卡上传/下载
- 点赞、评论、收藏系统
- 官方精选推荐
- 创作者激励计划

## 四、前 20 关难度曲线设计

- **渐进式学习：** 每关引入 1 个新概念
- **难度平滑：** 避免陡增，3 关一小台阶
- **重复巩固：** 新机制出现后，后续关卡复现强化
- **惊喜元素：** 每 5 关一个小高潮

场景：动态平台 + 多重机关 + 限时

## 六、Monetization（变现）策略

### 6.1 变现模型总览

采用 **混合变现模式**：IAP (内购) + IAA (广告)

```
                    收入结构
                        │
        ┌───────────────┼───────────────┐
        │               │               │
        ▼               ▼               ▼
    IAP 40%         IAA 35%        其他 25%
   (内购收入)      (广告收入)      (活动等)
```

### 6.2 内购设计 (IAP)

#### 6.2.1 商品分类

| 类型 | 商品 | 价格 (元) | 目标用户 |
|------|------|-----------|----------|
| 消耗品 | 体力 (5 点) | 3 | 全体 |
| 消耗品 | 体力 (25 点) | 12 | 中度玩家 |
| 消耗品 | 提示道具 (3 个) | 6 | 卡关玩家 |
| 消耗品 | 步数 +5 (3 个) | 6 | 挑战玩家 |
| 永久品 | 去广告 | 18 | 全体 |
| 永久品 | 双倍星星 | 25 | 收集玩家 |
| 订阅 | 周卡 (每日领体力) | 15/周 | 高频玩家 |
| 订阅 | 月卡 (每日领体力 + 道具) | 45/月 | 核心玩家 |

#### 6.2.2 定价策略

**新手礼包：** 首充 6 元 = 体力 30 + 提示 5 + 限定皮肤
**成长基金：** 累计付费返利，最高 10 倍回报
**限时特惠：** 节日/活动专属折扣

#### 6.2.3 付费点设计

```
付费触发点：
├── 体力耗尽 → 购买体力/看广告
├── 卡关多次 → 购买提示/看广告得提示
├── 追求完美 → 购买步数道具
├── 收集驱动 → 购买双倍星星
├── 体验优化 → 购买去广告
└── 长期游玩 → 订阅月卡
```

### 6.3 广告设计 (IAA)

#### 6.3.1 广告类型与位置

| 广告类型 | 触发场景 | 频次限制 | 激励内容 |
|----------|----------|----------|----------|
| 激励视频 | 体力恢复 | 5 次/天 | +5 体力 |
| 激励视频 | 获得提示 | 3 次/关卡 | 1 次提示 |
| 激励视频 | 步数 +5 | 3 次/关卡 | +5 步数 |
| 激励视频 | 双倍奖励 | 每关 1 次 | 结算奖励×2 |
| 插屏广告 | 关卡结束 | 每 3 关 1 次 | 无 |
| Banner | 主菜单底部 | 常驻 | 无 |

#### 6.3.2 广告体验优化

- **非强制：** 所有广告均为可选激励广告
- **频次控制：** 单用户单日广告不超过 15 次
- **内容审核：** 仅接入优质广告主
- **关闭选项：** 付费去广告
