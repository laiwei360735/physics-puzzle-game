# 🎮 物理益智小游戏 - 项目简报

## 项目概述
开发一款微信休闲小游戏，类似割绳子、倒水排序的物理益智游戏。

## 目标平台
- 微信小游戏
- 使用 Phaser 3 游戏引擎
- JavaScript/TypeScript

## 核心团队
| 角色 | Agent | 职责 |
|------|-------|------|
| 游戏策划 | agent-planner | 游戏设计文档、关卡设计 |
| 程序开发 | agent-dev | 核心代码、物理引擎集成 |
| 美术设计 | agent-art | 素材、UI、动画 |
| 测试 QA | agent-qa | 测试用例、bug 追踪 |

## 时间线
1. 策划阶段 - 游戏设计文档
2. 原型开发 - 核心玩法验证
3. 内容制作 - 关卡、美术
4. 测试优化 - Bug 修复、性能优化
5. 部署上线 - GitHub 发布

## 技术栈
- **引擎:** Phaser 3
- **语言:** TypeScript
- **物理:** Matter.js (Phaser 内置)
- **打包:** Webpack/Vite
- **微信适配:** 微信小游戏适配器
